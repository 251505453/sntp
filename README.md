sntp
====

### A implementation of NTP server with Golang

- 基于[NTP协议](http://tools.ietf.org/html/rfc2030)
- 其他设备从本机同步时间
- 适用于不能连接公网的局域网内设备的时间同步
- 严格遵循RFC2030协议，兼容linux下ntpdate服务
